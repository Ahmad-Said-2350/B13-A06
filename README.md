 DigiTools: Digital Tools Buying Platform.

 Project Description

DigiTools is a modern and fully responsive web application designed to help users explore and purchase premium digital tools in one place. From AI-powered writing assistants to developer tools, users can easily browse products, add them to the cart, and manage their selections with a smooth and interactive experience.

The project focuses on clean UI design, efficient state management, and real-world frontend development practices.



 Technologies Used

React.js
Tailwind CSS
DaisyUI
JavaScript (ES6+)
React-Toastify
JSON (for product data)



 Core Features

Add products to cart with a single click
Dynamic cart count displayed in the navbar
Remove individual items from the cart
Automatic total price calculation
Proceed to Checkout clears all cart items



 Product and Cart Toggle System

Two-button toggle system (Products and Cart)
Default view displays all products
Cart view shows selected items or an empty message



Real-time Notifications

Toast notifications for add to cart
Toast notifications for remove from cart
Toast notifications for checkout
Smooth and user-friendly feedback system
